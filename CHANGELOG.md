# Changelog

## [1.1.0] - 2025-03-31

### Added
- 新增 AB 性能对比面板，实时记录原生渲染和虚拟滚动的滚动耗时。
- 新增“运行性能测试”按钮，一键进行标准化性能对比。
- 新增虚拟滚动模式，基于固定节点池（默认 20 个 DOM 元素）实现真正的性能优化。
- 新增性能提升百分比计算，直观展示优化效果。

### Changed
- 优化 Pretext 库加载逻辑，确保测量前库已就绪。
- 改进设置面板 UI，整合所有控制项和统计信息。
- 修复高度管理器重建时的数据同步问题。

### Fixed
- 修复虚拟滚动启用时可能出现的 DOM 替换错误。
- 修复统计重置时性能数据未清零的问题。

## [1.0.0] - 2025-03-30

### Added
- 初始版本，提供 Pretext 高度测量和统计面板。
- 支持实时测量消息高度并输出性能日志。
- 提供测试工具，可测量任意文本的高度。
